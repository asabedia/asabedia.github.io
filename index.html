PocketCPM

An android application for project planning

Java, Kotlin, Dagger2, SQLite

Pocket CPM uses the Critical Path Method to process user provided data and provide key information that can be used to plan a project. By entering some basic information about a project and the tasks associated with it and their relationships, Pocket CPM is able to give you important dates for each task. This includes the earliest and latest start times as well as the earliest and latest finish times for each task. Pocket CPM is able to identify the critical path and denote slack times for individual tasks. In addition to providing the user with the ability to build a rich task tree, the user also has the ability to add and assign "Champions" to tasks who are individuals responsible for the completion of each task. Overall, Pocket CPM is a great tool for anyone looking to plan and approach a project in a structured manner.
